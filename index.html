FREQUENTLY ASKED QUESTIONS

How Does JavaScript works?

JavaScript is synchronous single threaded programming language.It is also known as scripting language.It runs inside the browser by javascript engine.Below are the steps of how javasript runs:

  • In every engine there is a parser which parse the javascript code.The parser checks the code whether it is ok or not.
  • After parsing it will create Abstract syntax tree.After AST the code is converted to machine code
  • After converting to machine code, then the code runs thus we get our output.
What is the difference between local storage and session storage?

Local storage and session storage is mostly similar. But there are some common differences which are mentioned below:

Local Storage Session Storage
Data remains permanently even if the domain is opened in a new tab, or the browser is closed and reopened Data is stored temporarily as long as the tab or the browser is open.
Data can be loaded again Data cant be laoded again
Has 5 mb limit Has 5-10 mb limit
Has Expiration Has no Expiration
